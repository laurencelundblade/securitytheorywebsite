QCBOR Documentation

QCBOR can be downloaded from its GitHub repository. This documentation corresponds to QCBOR v2, currently in its alpha release phase. While much of the content also applies to v1 – which is stable, commercially suitable, and compatible with v2 – it's important to note that v2 introduces new features and APIs not available in v1.
